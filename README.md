# Midnight Society - Playwright Automation Framework\n\nA comprehensive Playwright testing framework using the Page Object Model (POM) pattern for testing the Midnight Society website (midnightsociety.com).\n\n## Project Structure\n\n```\n├── pages/                    # Page Object Model classes\n│   ├── basePage.js          # Base class with common methods\n│   ├── homePage.js          # Home page object\n│   ├── navigationPage.js    # Navigation/Header page object\n│   ├── loginPage.js         # Login functionality\n│   ├── registrationPage.js  # Registration flows\n│   └── paymentPage.js       # Payment handling\n├── tests/                    # Test files (12 suites)\n├── playwright.config.js      # Playwright configuration\n├── package.json             # Project dependencies\n└── README.md               # This file\n```\n\n## Features\n\n✅ **Page Object Model Pattern** - Maintainable and scalable test structure\n✅ **Cross-browser Testing** - Chrome, Firefox, Safari, and mobile browsers\n✅ **Mobile Device Testing** - 9+ iOS and Android devices\n✅ **Responsive Testing** - 5 tablet and 5 desktop resolutions\n✅ **Security Testing** - HTTPS, XSS, SQL injection, CSRF\n✅ **Performance Testing** - FCP, LCP, CLS metrics\n✅ **Social Login Testing** - Google, Facebook, GitHub, Apple, Microsoft\n✅ **HTML Reports** - Detailed test reports with screenshots\n✅ **700+ Automated Tests** - Comprehensive coverage\n\n## Installation\n\n```bash\nnpm install\nnpx playwright install\n```\n\n## Running Tests\n\n```bash\nnpm test                 # Run all tests\nnpm run test:headed      # Run with visible browser\nnpm run test:ui          # Run with Playwright UI\nnpm run test:chrome      # Chromium only\nnpm run report           # View HTML report\n```\n\n## Test Suites\n\n- **Home Page** - Homepage load, header, navigation (11 tests)\n- **Login** - Form validation, password masking (10 tests)\n- **Registration** - Signup flow, field validation (11 tests)\n- **Payment** - Card validation, checkout flow (20 tests)\n- **Security** - HTTPS, XSS, SQL injection, etc (15 tests)\n- **UI Elements** - Responsive, color contrast, fonts (16 tests)\n- **Performance** - Load time, FCP, LCP, CLS (13 tests)\n- **Edge Cases** - Long inputs, Unicode, timeouts (20 tests)\n- **E2E Flows** - End-to-end user journeys (25 tests)\n- **Social Login** - OAuth flows and SSO (11 tests)\n- **UI Comprehensive** - 50+ UI element tests\n- **Responsive Devices** - 100+ device/resolution tests\n\n## Mobile Device Coverage\n\n- iPhone 16/16 Pro, iPhone 17/17 Pro\n- Pixel 8, Pixel 9\n- Galaxy S24, OnePlus 12, Xiaomi 14\n- iPad Air, iPad Pro (11\" & 12.9\")\n- Samsung Galaxy Tab S9, Microsoft Surface Go\n\n## Desktop Resolutions\n\n- 1024px (Small), 1366px (HD), 1920px (Full HD)\n- 2560px (Large), 3440px (Ultra Wide)\n\n## Support\n\nFor issues, check the [Playwright Documentation](https://playwright.dev)
